색 이름
가장 쉬운 방법으로는 색깔에 이름을 붙이는 방식이 있습니다.
물감의 빨강, 초록, 노랑 같이 CSS에서 사용하는 색에도 이름이 있는데요.
red, green, yellow 같은 영어 이름이 있습니다.
색 이름은 자주 사용하는 방법은 아니니까 참고로만 알아 두시면 됩니다.
어떤 이름이 있는지 궁금하신 분들은 CSS 표준 문서의 컬러 키워드를 참고해 주세요.

색상 코드
색 이름은 하나하나 붙이기도 힘들고 색을 섬세하게 나타내기 어렵습니다.
그래서 보통은 색상 코드라는 걸 사용하는데요.
구글에서 'Color Picker'라고 검색하면 아래 같은 색상 선택 도구가 나옵니다.
구글 컬러 피커
보시면 HEX(16진수)라는 게 있는데요. 이게 색상 코드입니다.
샵(#)을 붙이고 뒤에 여섯 글자로 된 코드입니다.
간단히 말해서 모든 색깔들을 쭉 펼쳐놓고 하나씩 코드를 붙여 놓은 건데요.
예를 들어서 빨간색은 #ff0000, 파란색은 #0000ff 이런 식으로 나타낼 수 있습니다.
주로 이미지 편집 프로그램 같은 데서 디자인을 하면서 색상 코드를 정하게 됩니다.
참고로 색상 코드는 대소문자를 구분하지 않는데요, 그래서 #ff0000과 #FF0000은 똑같은 빨간색입니다.
display에는 여러가지 속성이 있는데요 none block inline inline-block flex 등등
visibility에도 visible hidden collapse 등의 속성이 있어요 
문제나갑니다 display none과 visibility의 hidden 의 차이점을 설명해주세요
